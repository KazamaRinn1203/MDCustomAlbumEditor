Editing Tools

Double-click empty lane space to create a note
Marquee-select multiple notes for batch edits
Double notes render on both lanes while exporting from one source note
Spinner / burst behaves like a sustained segment with start/end markers
Cover export supports zoom, pan, reset framing, and circular masking

What It Solves

Waveform-Based Authoring

Lets users place notes against decoded song audio instead of manually typing time values.

Better Editor Ergonomics

Improves tapping flow, keyboard controls, drag thresholds, double-note visualization, and batch editing.

Asset Preparation in the Same Tool

Generates cover art and demo clips without forcing users into separate image and audio tools.

Game-Ready Packaging

Builds the final export package with the files needed for a Muse Dash custom album mod workflow.

Workflow

Import

Users bring in an MP3, which is converted into an internal OGG source for editing and export.

Author

Notes are placed and adjusted on a waveform timeline, including hold, double, and spinner/burst behaviors.

Package

Demo audio, circular cover art, metadata, and BMS output are assembled into a final .mdm package.

Engineering Highlights

Desktop Architecture

Used Electron so the editor could work directly with local audio files, cover images, and package output.

Interaction Iteration

Refined note creation, dragging, keyboard flow, selection models, and export UX around real creator feedback.

Production-Oriented Tooling

Focused on reducing friction across the full pipeline instead of only editing notes in isolation.
