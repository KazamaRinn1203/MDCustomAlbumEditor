Muse Dash Custom Album Editor

A desktop chart editor for Muse Dash custom content, focused on speeding up note authoring, asset preparation, and packaging into game-ready .mdm files.

Electron JavaScript FFmpeg Audio Tooling Desktop UX

What It Does

Waveform Charting

Places and edits notes against audio timing with keyboard tapping, snapping, and lane-based authoring.

Authoring Tools

Supports tap, hold, double, and spinner/burst notes, plus batch editing through multi-selection.

Asset Generation

Auto-converts imported audio, generates timeline-based demo clips, and builds circular transparent cover art.

One-Click Packaging

Exports a full `.mdm` package including music, demo, cover, metadata, and BMS content.

Project Impact

Desktop

Built as an Electron application for direct file-system access and local media workflows.

End-to-End

Covers import, authoring, preview, asset prep, and export in a single tool.

Tooling UX

Iterated around practical creator pain points like BPM detection, demo selection, and cover framing.
