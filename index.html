Lane Editor

Tapped notes start on lane 1. Click notes directly on the lanes to define and move them.

Tapping Guide

  • Press Z or X while audio plays to create a note.
  • Both keys create notes on the same timing lane.
  • Double-click empty lane space to create a note at that position.
  • Click the waveform to seek.
  • Select a note and press Delete or Backspace to remove it.
